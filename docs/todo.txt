Boot up
=======
Obtain memory map.
Find space for kernel stack.
Set up kernel stack.
Find space for kernel code+data.
Load kernel into memory.
Enter protected mode.
Enter long mode.

Physcial memory
===============
Create physical memory allocator.

Physical memory layout plan is:


:        ....        :
|       kernel       |
|--------------------| 0x100000
|                    |
:        ....        :
|                    |
|--------------------| 
| bootloader phase 2 |
|--------------------| 0x7d00
| bootloader phase 1 |
| -------------------| 0x7c00
|  bootloader stack  |
:        ....        :
